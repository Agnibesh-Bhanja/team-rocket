# ⚡ Team Rocket Intel System

A Pokémon-inspired React web application where users can upload, view, and manage Pokémon intel.

Built with React, Supabase, and deployed on Vercel.

---

## 🚀 Features

- 🔐 User Authentication (Supabase Auth)
- 📝 Create Pokémon Intel Posts
- 🖼️ Image Upload with Supabase Storage
- 🎵 Background Music on Interaction
- ⚡ Logo Shock Animation
- 🔴 Purify Mode (Red Theme Toggle)
- 🎨 Dynamic UI with Animated Pokéball
- 🌐 Deployed on Vercel

---

## 🛠️ Tech Stack

- React (Vite)
- React Router
- Supabase (Database + Auth + Storage)
- CSS (Custom Styling + Animations)
- Vercel (Deployment)
